HOP is Open HWP

오픈소스 HWP 앱

HOP는 HWP/HWPX 문서를 보고 편집할 수 있는 오픈소스 macOS, Windows, Linux용 데스크톱 앱입니다.

GitHub 모든 릴리즈

최신 버전 다운로드

macOS Apple Silicon .dmg macOS Intel .dmg Windows x64 .msi Linux x64 AppImage

macOS는 signed/notarized 빌드입니다. Windows는 아직 서명되지 않아 실행 시 보안 경고가 뜰 수 있습니다.

Windows에서 "일반적으로 다운로드되지 않습니다"가 표시되면 다운로드 항목의 ... 메뉴에서 유지를 선택한 뒤 다운로드해 주세요. 이 경고는 서명되지 않은 새 설치 파일에서 주로 발생합니다.

HOP는 rhwp를 기반으로 합니다. HWP 엔진을 공개해 주신 개발자분께 감사드립니다.
