HOP is Open HWP

오픈소스 HWP 앱

HOP는 HWP/HWPX 문서를 보고 편집할 수 있는 오픈소스 macOS, Windows, Linux용 데스크톱 앱입니다.

최신 버전 다운로드

최신 릴리즈 파일로 연결됩니다.

macOS

Apple Silicon .dmg Intel .dmg

Homebrew로도 설치할 수 있습니다. brew install hop
버전이 자주 업데이트되므로, 주기적으로 최신 버전으로 업그레이드 해 주세요. brew upgrade hop

Windows

x64 .msi x64 .exe setup

아직 서명되지 않아 다운로드나 실행 중 보안 경고가 표시될 수 있습니다. Edge에서 "일반적으로 다운로드되지 않습니다"가 표시되면 다운로드 항목의 ... 메뉴에서 유지를 선택해 주세요.

Linux

x64 .deb (권장) Debian/Ubuntu x64 .rpm Fedora/openSUSE AppImage portable

한글 IME 안정성을 위해 배포판에 맞는 .deb 또는 .rpm 패키지를 먼저 사용해 주세요. AppImage는 별도 설치 없이 실행해야 할 때 적합합니다.

HOP는 rhwp를 기반으로 합니다. HWP 엔진을 공개해 주신 개발자분께 감사드립니다.
